    Subscribe API methods¶

    PaymeSubscribeCards¶

    The PaymeSubscribeCards class includes all paycom methods which are belongs to cards

    cards_create()¶

    To create a new card's token

    Argument Type Description
    number str Required. The card number maximum length 18 char
    expire str Required. The card expiration string maximum length 5 char
    save boolean Optional. By default it equals true. Type of token. The option is enabled or disabled depending on the application's business logic. If the flag is true, the token can be used for further payments. If the flag is false the token can only be used once. The one-time token is deleted after payment

    Example to cards create method 

    from pprint import pprint

from payme.cards.subscribe_cards import PaymeSubscribeCards

client = PaymeSubscribeCards(
    base_url="https://checkout.test.paycom.uz/api/",
    paycom_id="your-paycom-id"
)

response = client.cards_create(
    number="8600069195406311",
    expire="0399",
    save=True
)

pprint(response)
